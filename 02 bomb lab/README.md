# 02 Bomb Lab

## 文件内容

- 项目文件：`bomb` 
- 相关说明教程：`Instruction`
- 其余：
    - `bomb.txt` 为反汇编内容

```sh
bomb/bomb ans.txt
```

#### 补充

```c
// func4 对应的C语言代码
// 转自https://zhuanlan.zhihu.com/p/104130161

int func4(int x, int y, int z) {
    int k = z - y;
    k = ((int)(((unsigned)k>>31) + k) >> 1) + y;
    if (k < x)
        return 2 * func4(x, k + 1, z) + 1;
    else if (k > x)
        return 2 * func4(x, y, k - 1);
    else
        return 0;
}
```

因此第一个数还可以是 `0` / `1` / `3`

- 第19~32行是一个二重循环，要求这6个数互易
